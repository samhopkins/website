Outreach

Thinking about programming language design and an introduction to web programming (for Lakeside School computer science). For students who have only seen Java, an introduction to web programming is a great forum to think for the first time about differences between programming languages and what goes into the design of a programming language, and it allows for students with basic programming experience to get something working quickly in a new language. I talk about Dart because it's what I know well and it's easy for Java-only students to pick up, but Javascript would probably work equally well.
